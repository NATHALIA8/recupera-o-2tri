Comandos HTML e CSS

Descubra os comandos essenciais para construir e estilizar sites

Comandos HTML

<!DOCTYPE html>

Declara o tipo de documento e informa ao navegador como renderizar a página.

<html>

Elemento raiz do documento HTML. Todo o conteúdo é envolvido por esta tag.

<head>

Contém metadados, links para CSS e outros recursos importantes.

<body>

Contém o conteúdo visível da página, incluindo texto e imagens.

<a>

Cria links para outras páginas ou recursos. Exemplo: <a href="link">Texto</a>.

<img>

Insere imagens na página. Exemplo: <img src="imagem.jpg" alt="Descrição">.

<div>

Usado para agrupar e estilizar blocos de conteúdo.

<p>

Define um parágrafo de texto. Exemplo: <p>Texto</p>.

<h1> a <h6>

Define cabeçalhos de diferentes níveis. Exemplo: <h1>Cabeçalho</h1>.

<span>

Usado para estilizar partes do texto dentro de um bloco. Exemplo: <span style="color:red">Texto</span>.

<ul>, <ol>, <li>

Define listas não ordenadas e ordenadas. Exemplo: <ul><li>Item</li></ul>.

Comandos CSS

color

Define a cor do texto. Exemplo: color: red;.

background-color

Define a cor de fundo de um elemento. Exemplo: background-color: blue;.

font-size

Define o tamanho da fonte. Exemplo: font-size: 20px;.

margin

Define a margem ao redor de um elemento. Exemplo: margin: 10px;.

padding

Define o espaçamento interno de um elemento. Exemplo: padding: 15px;.

border

Adiciona uma borda ao redor de um elemento. Exemplo: border: 1px solid black;.

display

Define como um elemento é exibido na página. Exemplo: display: flex;.

position

Controla o posicionamento de um elemento. Exemplo: position: absolute;.

width e height

Define a largura e altura de um elemento. Exemplo: width: 100px; e height: 50px;.

flexbox

Layout flexível para criar layouts responsivos. Exemplo: display: flex;.

grid

Layout em grid para criar layouts complexos. Exemplo: display: grid;.
